# MCQquiz

APIs used 

Faculty:                           
  1.create quiz          
  2.add question to quiz              
  3.create student for test                
                  
Student:                      
  1.login                      
  2.get list of test which can appear                
  3.start that test                       
                                              
********* To Run This Application**********                      
1.Download or clone this repo                
2.run npm install                           
3.configure .env file as per your environment.                           
4.npm start
